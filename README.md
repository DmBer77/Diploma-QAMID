## Процедура запуска автотестов:

#### 1. Предварительные условия
   - тесты написаны в Intellij IDEA при помощи Appium, по этому в компьютере должен быть установлен Appium

#### 2. Подключение устройства и запуск программ
   - запустить Appium server
   - запустить Intellij IDEA
   - подключить физическое устройство к компьютеру (или запуск эмулятора) с установленным тестовым приложением

#### 3. Запуск тестов
   - перейти в папку: src/test/java/ru.netology/qa
   - открыть по очереди все 4 файла в папке (тесты логически разбиты по разделам тестируемого приложения)
   - в каждом файле нажать на иконку запуска тестов (либо групповой запуск (первая иконка), либо каждый тест по отдельности)
